Аннотация проекта

Проект "ИТ-сервисы для Цифрового университета" направлен на создание современной цифровой экосистемы для Московского Политеха. Мы разрабатываем комплекс решений, включающий мобильные приложения, аналитические сервисы и системы автоматизации учебных процессов.

Мы стремимся повысить эффективность образовательного процесса за счёт внедрения цифровых технологий, улучшить взаимодействие между студентами и преподавателями, а также создать удобную платформу для управления учебной деятельностью.

Цели нашей команды
Разработать удобное мобильное приложение личного кабинета
Задачи
  • разработать дизайн для новых модулей
  • определить части проекта, нуждающиеся в доработке до релиза android-версии
  • исправить все найденные недоработки
  • изменить интерфейс приложений до полного соответствия с разработанным дизайном
  • провести тестирование приложений, исправить баги
